# Quick Start Guide - System Design Agent

## Get Started in 3 Steps

### Step 1: Open the App
```
http://localhost:3333
```

### Step 2: Enable Demo Mode
Look for the **Demo Mode** toggle switch and turn it ON (it will turn teal)

### Step 3: Generate a Design
Click one of the example buttons or type your own requirements:

**Try this:**
```
E-commerce platform handling 100K concurrent users with payment processing, inventory management, and PCI-DSS compliance
```

Then click **"Generate Demo Design"**

---

## What You'll See

Within 3 seconds, you'll get:
- Interactive architecture diagram with 8 components
- Clickable components showing technologies
- Trade-off analysis
- Validation report
- Cost estimation

## Try These Features

### Edit a Component
1. Click **"Edit Mode"** button (top right)
2. Click any component (blue badges appear)
3. Edit name, type, purpose, technologies, etc.
4. Click **"Save Changes"**

### Export Report
1. Click **"Export Report"** button (top right)
2. JSON file downloads automatically
3. Contains complete design with all details

### View Trade-Offs
Click **"Trade-Offs"** button to see:
- Architecture decisions
- Benefits vs costs
- Mitigation strategies

### View Validation
Click **"Validation"** button to see:
- Potential issues
- Severity levels
- Recommended mitigations
